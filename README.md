# Whisker Wealth

A cozy kitty-themed idle tycoon game. Build cat businesses, hire managers, buy upgrades, and chase purrposterous profits.

## Play

Open `index.html` in a browser, or publish with GitHub Pages.

## GitHub Pages setup

1. Go to **Settings → Pages**.
2. Under **Build and deployment**, choose **Deploy from a branch**.
3. Select branch: `main`.
4. Select folder: `/root`.
5. Save.

The live game should publish at:

`https://dkruse-rgb.github.io/Whiskerwealth/`

## Files

- `index.html` — the full playable game.
- `.nojekyll` — helps GitHub Pages serve the static file cleanly.
